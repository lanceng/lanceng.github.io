Mongo DB介绍

  • mongodb数据 nodejs
database

Mongo DB ,是目前在IT行业非常流行的一种非关系型数据库(NoSql),其灵活的数据存储方式,备受当前IT从业人员的青睐。Mongo DB很好的实现了面向对象的思想(OO思想),在Mongo DB中 每一条记录都是一个Document对象。Mongo DB最大的优势在于所有的数据持久操作都无需开发人员手动编写SQL语句,直接调用方法就可以轻松的实现CRUD操作。

Mongo DB安装

下载地址：http://www.mongodb.org/downloads

安装版本分为Windows、Linux、Mac,根据自己的需要下载安装。

Mongo DB的基本操作

1、新建集合级：

db.createCollection("user");

2.插入数据

db.user.insert({uid:1,username:"Jerry",age:20});   
j = {name:"mongo"};   
db.things.save(j);   
for(var i = 1; i < 10; i++) db.tables.save({x:4,j:i});

3.更新数据：

db.user.update({uid:1},{$set:{age:23}}) #age=26;   
db.user.update({uid:1},{$inc:{age:-1}}) #age=age-1;

4.查询：

4.1 遍历集
var cursor = db.things.find();
while (cursor.hasNext()) {print(tojson(cursor.next()));

或 

db.things.find().forEach( function(x){print(tojson(x)); });

4.2 获取结果集 

var cursor =db.things.find();   
print (tojson(cursor[4]);  
var arr = db.things.find().toArray();   
arr[5];

4.3 条件查询 

db.things.find({name:"mongo"}).forEach(function(x){ print(tojson(x)); });

等价于：
SQL: SELECT * FROM things WHERE name = “mongo”; 

db.things.find({x:4,{j:true}).forEach(function(x){ print(tojson(x)); {);

等价于：
SQL:SELECT J FORM things WHERE X = 4;

4.4 sort用法： 

db.things.find({tags:'economy'}).sort({ts:-1}).limit(10);

等价于：
SQL:SELECT * FROM things WHERE ‘economy’ IN tags ORDER BY ts DESC LIMIT 10;

4.5 findOne用法 

var mongo = db.things.findOne({name:"mongo"});   
print(tojson(mongo));

4.6 limit用法： 

db.things.find().limit(3);
