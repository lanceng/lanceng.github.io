JAVA对Excel的导入导出操作

  • excel操作
JAVA

在JAVA中对Excel的操作有以下几种方式：POI方式、JXL方式、FastExcel方式。在JAVA中对pdf的操作，比较流行的是用iText方式操作。

一、Exce解析的几种方式

1、POI方式

HSSF是Horrible SpreadSheet Format缩写，中文：“讨厌的电子表格式”。通过HSSF，可以用纯粹的JAVA代码来实现对Excel文件的读取、写入、 修改操作。

HSSF - 读写Microsoft Excel格式文档的功能
XSSF - 读写Microsoft Excel OOXML格式文档的功能
HWPF - 读写Microsoft word格式文档的功能
HSLF - 读写Microsoft PowerPint格式文档的功能
HDGF - 读写Microsoft Visio格式文档的功能

2、JXL方式

JAVA Excel是一个开源项目，可以读取Excel文件的内容、创建新的Excel文件、更新已经存在Excel文件。

包括常见的格式设置：字体、颜色、背景、合并单元格等。

3、FASTEXCEL方式

FastExcel是一种采用纯JAVA开发的excel读写组件，支持excel 97-2003格式文件。
FastExcel只能读取单元格的文字信息，而其他属性，如颜色、字体就不能够支持了，因此FastExcel所需的内存很小。

4、iText

通过iText不仅可以生成PDF或rtf文件，还可以将XML、HTML格式文件转换成PDF文件。

下载iText.jar文件后，只要在系统的CLASSPATH中加入iText.jar的路径，就可以在程序中使用iText库类了。

二、JXL详解

使用JXL进行Excel的创建和读取需要用到jxl.jar包，下载将此包导入即可。
API地址：http://www.andykhan.com/jexcelapi/tutorial.html

1、创建一个工作簿 

WritableWorkbook wwb = Workbook.createWorkbook(os);

其中os为一个文件输出流。当然还有很多其他的入参，比如File等。

2、读取现有的工作薄 

Workbook.getWorkbook(java.io.File file);

3、创建工作表Sheet 

WritableSheet sheet = wwb.createSheet("职员表", 0);

4、创建标签Label 

Label label = new Label(col, row, title);

将标签加入到工作表中： 

sheet.addCell(label);

5、填充数据 

jxl.write.Number numb = new jxl.write.Number(1, 1, 250); 
sheet.addCell(numb);

格式化数字 

jxl.write.NumberFormat nf = new jxl.write.NumberFormat("#.##"); 
jxl.write.WritableCellFormat wcf = new jxl.write.WritableCellFormat(nf); 
jxl.write.Number n = new jxl.write.Number(2, 1, 2.451, wcf); 
sheet.addCell(n);

填充日期 

SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd hh:mm:ss"); 
String newdate = sdf.format(new Date()); 
label = new Label(2, 2, newdate); 
sheet.addCell(label);

填充文本 

label = new Label(3, 3, "周星驰"); 
sheet.addCell(label);

填充boolean值 

jxl.write.Boolean bool = new jxl.write.Boolean(4, 1, true); 
sheet.addCell(bool);

6、合并单元格

通过writablesheet.mergeCells(int x,int y,int m,int n);来实现的。表示将从第x+1列，y+1行到m+1列，n+1行合并 (四个点定义了两个坐标，左上角和右下角)，结果是合并了m-x+1行，n-y+1列，两者乘积就是合并的单元格数量。 

sheet.mergeCells(0, 6, 3, 8); 
 label = new Label(0, 6, "合并了12个单元格"); 
sheet.addCell(label);

7、添加单元格样式

主要是设置单元格的背景、字体、样式等 

WritableCellFormat wc = new WritableCellFormat(); 
// 设置居中 
wc.setAlignment(Alignment.CENTRE); 
// 设置边框线 
wc.setBorder(Border.ALL, BorderLineStyle.THIN); 
// 设置单元格的背景颜色 
wc.setBackground(jxl.format.Colour.RED); 
label = new Label(1, 5, "字体", wc); 
sheet.addCell(label);

8、设置单元格字体 

// 设置字体 
jxl.write.WritableFont wfont = new jxl.write.WritableFont(WritableFont.createFont("楷书"), 20); 
WritableCellFormat font = new WritableCellFormat(wfont); 
label = new Label(2, 6, "楷书", font); 
sheet.addCell(label);

9、将工作簿写成文件 

// 写入数据 
wwb.write(); 
// 关闭文件 
wwb.close();

10、对行列的批量操作 

//获取所有的工作表 
jxl.write.WritableSheet[] sheetList = wwb.getSheets(); 
//获取第1列所有的单元格 
jxl.Cell[] cellc = sheet.getColumn(0); 
//获取第1行所有的单元格 
jxl.Cell[] cellr = sheet.getRow(0); 
//获取第1行第1列的单元格 
Cell c = sheet.getCell(0, 0);

11、获取单元格的值 

//获取单元格的值，不管什么单元格，返回都是字符串 
String value = c.getContents();

三、POI方式详解

首先，上apache poi官方文档连接：http://poi.apache.org/spreadsheet/quick-guide.html

1、读取excel文件 

BufferedInputStream bis = new BufferedInputStream(new FileInputStream(filePath));
POIFileSystem fs = new POIFileSystem(bis);
HSSFWorkbook sampleWorkbook = new HSSFWorkbook(fs);
HSSFSheet sampleSheet = sampleWorkbook.getSheet(0);//获取第一个工作簿

2、创建一个工作簿 

HSSFWorkbook book = new HSSFWorkbook();
HSSFSheet sheet = book.createSheet(fileName);
sheet.setDisplayGridlines(false);//这里是设置excel文件是否显示默认的网格线，为true时是默认值显示，false时是不显示。

3、创建行、列、设置行宽、列宽 

sheet.setColumnWidth(0, 5000);//这里是设置第一列的列宽为5000
HSSFRow sampleRow = sampleSheet.getRow(0);//获取模板文件的第一行
HSSFRow row = sheet.createRow(0);//创建第一行
HSSFCell sampleCell = sampleRow.getCell(0);//获取模板文件第一行第一列
HSSFCell cell = row.createCell(0);//创建第一行第一列

row.setHeight(5000);//设置第一行行宽
sheet.addMergedRegion(newCellRangeAddress(firstc, firstr, lastc, lastr));//合并n行n列

int startNum = sampleRow.getFirstCellNum();//获取模板行的开始列索引
int endNum = sampleRow.getLastCellNum();//获取模板行的结束列索引

4、样式设置 

HSSFCellStyle style = book.createCellStyle();//创建一个单元样式

HSSFFont foot = book.createFont();//创建一个字体样式
font.setFontName("宋体");
font.setFontHeightInPoints((short)24);

style.setFont(font);
style.setAlignMent(HSSFCellStyle.ALIGN_RIGHT);//文本位置
style.setFillBackgroundColor(HSSFColor.GRY_25_PERCENT.index);//背景色设置
style.setFillForegroundColor(HSSFColor.GREY_25_PERCENT.index);//前景色设置
style.setFillPattern(HSSFCellStyle.FINE_DOTS);//填充色
style.setBorderLeft((short)1);//左边框大小
style.setBorderRight((short)1);//右边框大小
style.setBorderBottom(HSSFCellStyle.BORDER_THIN);//边框为粗体
style.setBottomBorderColor(HSSFColor.BLACK.index);//边框颜色

style.cloneStyleFrom(sampleCell.getCellStyle());//复制样式

cell.setCellStyle(style);//设置单元格样式

5、导出excel文件 

private void organizeResponse(HttpServletResponse response, HSSFWorkbook workbook, String begindate, String enddate){
OutputStream out = null;
try{
    out = response.getOutputStream();
    response.reset();
    String filename = java.net.URLEncoder.encode("基金销售机构交易数据报送表_"+enddate+".xls", "UTF-8");
    response.addHeader("Content-Disposition", "attachment; filename=" + filename);
    response.addHeader("charset", "ISO8859-1");
    response.setContentType("application/octet-stream");
    workbook.write(out);
    out.flush();
}catch (Exception e){
    e.printStackTrace();
}finally {
    try{
        if(out != null)out.close();
    }catch (Exception e){
        e.printStackTrace();
    }
}
}
